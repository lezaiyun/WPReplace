== Description ==

<strong>WordPress内容字符替换插件（简称:WPReplace），可视化帮助网友快速替换WordPress网站文章标题、内容、评论用户名和评论内容的指定字符。站长交流QQ群： <a href="https://jq.qq.com/?_wv=1027&k=5gBE7Pt" target="_blank"> <font color="red">594467847</font></a></strong>

<strong>主要功能：</strong>

* 1、下载和激活【WPReplace】插件后，我们就可以在后台菜单看到工具操作界面
* 2、目前支持网站内容、标题、评论指定字符的快速替换
* 3、可以快速解决以往我们需要从MYSQL数据库命令替换的烦恼
* 4、WPReplace插件更多详细介绍和安装：<a href="https://www.laobuluo.com/2693.html" target="_blank" >https://www.laobuluo.com/2693.html</a>

<strong>支持网站平台：</strong>

* 1. 老蒋部落 <a href="https://www.itbulu.com" target="_blank" >https://www.itbulu.com</a>
* 2. 老部落 <a href="https://www.laobuluo.com" target="_blank" >https://www.laobuluo.com</a>
* 3. 推荐文章：<a href="https://www.laobuluo.com/2113.html" target="_blank">新人建站常用的虚拟主机/云服务器 常用主机商选择建议</a>

== Installation ==

* 1、把WPUPYUN文件夹上传到/wp-content/plugins/目录下<br />
* 2、在后台插件列表中激活WPUPYUN<br />
* 3、在左侧【WP又拍云设置】菜单中输入又拍云存储空间账户信息。<br />
* 4、设置可以参考：https://www.laobuluo.com/2620.html

== Frequently Asked Questions ==

* 1.当发现插件出错时，开启调试获取错误信息。
* 2.我们在使用WPReplace插件之前记得先备份数据库，确保数据安全。
* 3.WPReplace插件可能会和有些未知插件冲突。
* 4.如果不熟悉使用这类插件的用户，一定要先备份，确保错误设置导致网站故障。
